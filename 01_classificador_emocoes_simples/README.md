# Classificador de sentimentos (Simples)

Exercío de Machine Learning criado para praticar conceitos fundamentais de processamento de texto e modelos de classificação utilizando a biblioteca scikit-learn.

O objetivo é treinar um modelo capaz de classificar frases curtas como positivas ou negativas, usando um conjunto pequeno de frases previamente rotuladas.

## Objetivo
- Fixar conhecimentos iniciais de Machine Learning aplicado a texto (NLP)
- Entender o fluxo básico: dados → vetorização → treinamento → predição
- Criar um exemplo simples e funcional de classificação de sentimentos

## Tecnoligias
- Python
- scikit-learn
    - CountVectorizer para transformar texto em vetores numéricos
    - MultinomialNB para o modelo de classificação Naive Bayes

## Como Executar

1. Ativar o ambiente virtual

`venv\Scripts\Activate`

2. Rodar o script:

`python classificador.py`

O script exibirá no terminal a classificação das frases de teste (positivas ou negativas).
