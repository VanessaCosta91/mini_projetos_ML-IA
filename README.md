# Projetos Machine Learning e IA: Repositório de Estudos e Evolução

Este repositório foi criado com o objetivo de registrar minha evolução prática enquanto estudo técnicas de ML, IA, NLP e Recomendação de Conteúdo.

Aqui você encontrará projetos curtos, objetivos e voltados para aprendizagem contínua. Cada pasta representa um projeto independente, contendo:

- código-fonte
- dados ou links para datasets
- README específico do projeto
- resultados, métricas e anotações de aprendizado

*Este repositório será atualizado constantemente conforme novos projetos forem sendo finalizados.*

---

## Objetivo do Repositório

Criar um portfólio sólido e prático de Machine Learning que demonstre:

- domínio de pré-processamento e tratamento de dados
- aplicação de algoritmos clássicos de ML
- experimentos com NLP e análise de texto
- implementação de modelos de recomendação
- primeiros passos em IA generativa, embeddings e RAG

Além disso, os projetos aqui servem como base para meu projeto principal em desenvolvimento:
Agente Conversacional para Sugestão Personalizada (LLM + RAG).
